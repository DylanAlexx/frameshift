# FrameShift

An open-source media analysis platform for local video libraries.

## Vision

FrameShift analyzes video, audio, and subtitle data to build reusable fingerprints that enable features such as intro detection, credit detection, duplicate detection, and more.

## Status

🚧 Early development

## Development

Install dependencies:

```bash
uv sync
```

Run the formatter:

```bash
just format
```

Run linting:

```bash
just lint
```

Run tests:

```bash
just test
```

Run all checks:

```bash
just check
```
